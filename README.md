Fitur-fitur utamanya:

1. Dukungan Dual Metode: Bisa untuk Media Tanah (Jadwal siram & pupuk) dan Hidroponik (Cek air, nutrisi, kuras tandon).

2. Manajemen Koleksi Tanaman: Detail tanaman lengkap dengan foto, tanggal tanam, dan kategori.

3. Kalkulator Nutrisi: Fitur hitung dosis AB Mix presisi berdasarkan kalibrasi PPM (sangat berguna untuk hidroponik).

4. Analisis Sensor: Input data manual atau sensor (Suhu, pH, Kelembaban, Lux, EC) untuk mendapatkan analisis kondisi tanaman.

5. Klinik Tanaman: Mencatat kesehatan tanaman, diagnosa hama/penyakit, dan solusi yang dilakukan.

6. Manajemen Keuangan: Mencatat pemasukan (panen) dan pengeluaran (bibit/pupuk) serta grafik hasil panen.

7. ID Card Tanaman: Membuat QR Code unik untuk setiap tanaman.

8. Backup & Restore: Data bisa diekspor ke Excel atau JSON.
